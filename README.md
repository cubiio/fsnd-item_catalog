### Vagrant

Move to the project folder:
```
$ cd /vagrant/fsnd-item_catalog
```

Fire up Vagrant:
```
# load vagrant
$ vagrant up

# login via ssh
$ vagrant ssh
```

Once Vagrant is up:
```
# move to the vagrant folder:
$ cd /vagrant

# move to the project folder:
$ cd fsnd-item_catalog

# or 
$ cd /vagrant/fsnd-item_catalog
```

To exit/stop Vagrant:
```
$ exit
$ vagrant halt
```
